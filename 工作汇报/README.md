# 工作汇报资料

本目录用于存放 2026 年工作汇报相关材料，包括原始文本、PPT 成品、Keynote 兼容版本、生成脚本和提示词。

## 文件说明

- `2026年工作汇报.txt`：工作汇报原始文本内容。
- `2026年工作汇报.pptx`：工作汇报 PPT 文件。
- `2026年上半年工作汇报_Keynote兼容版.pptx`：Keynote 兼容版本。
- `2026年上半年工作汇报_浅色Keynote兼容版.pptx`：浅色风格 Keynote 兼容版本。
- `generate_work_report_ppt.py`：PPT 生成脚本。
- `PPT生成Prompt.md`：用于生成或优化 PPT 的提示词。
