# 本体论元数据 AI 模型资料

本目录用于沉淀“本体论如何在元数据层落地，并形成 AI 可理解的模型数据”的相关内容。

## 文件说明

- `本体论在元数据层的落地方案.md`：面向方案汇报、项目讨论和后续建设的主文档。
- `数据开发助手如何基于本体论快速查找元数据.md`：说明数据开发助手如何通过本体论实现找表、找字段、找指标、找关系和找血缘。
- `基于本体论的数据资产盘点与元数据自动补全设计方案.md`：说明如何用本体论盘点数据资产，并接入元数据字段、中文名、业务含义和备注信息的自动补全。
- `一键智能填充原型/index.html`：基于现有盘点页面制作的一键智能填充可交互高保真原型。
- `一键智能填充原型/Axure原型规格.md`：Axure 页面、动态面板、元件命名、中继器字段和交互用例规格。
- `一键智能填充原型/数据资产盘点一键智能填充原型.drawio`：可在 draw.io 中直接打开和编辑的四页面原型图。
- `一键智能填充原型/数据资产盘点一键智能填充原型.rp`：由 Axure RP 11 生成并验证可打开的单页四状态原型工程；最新对齐版以 draw.io、SVG 和四张 PNG 为准。
- `一键智能填充原型/01_一键填充入口.png` 至 `04_生成结果与依据.png`：用于导入 Axure RP 11 的四张页面素材。
- `一键智能填充原型/原型四步骤母版.svg`：采用固定坐标栅格生成的四步骤矢量母版，用于保证标签、冒号和字段列精确对齐。
- `ontology_metadata_model_example.json`：可机器读取的语义模型样例，展示实体、属性、关系、规则、血缘和权限如何结构化表达。

## 使用场景

- 数据治理方案设计
- 元数据管理平台建设
- 指标体系和语义层建设
- 智能问数、Text-to-SQL、RAG 和知识图谱应用
- 面向 AI 的企业数据资产梳理
