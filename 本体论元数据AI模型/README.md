# 本体论元数据 AI 模型资料

本目录用于沉淀“本体论如何在元数据层落地，并形成 AI 可理解的模型数据”的相关内容。

## 文件说明

- `本体论在元数据层的落地方案.md`：面向方案汇报、项目讨论和后续建设的主文档。
- `数据开发助手如何基于本体论快速查找元数据.md`：说明数据开发助手如何通过本体论实现找表、找字段、找指标、找关系和找血缘。
- `基于本体论的数据资产盘点与元数据自动补全设计方案.md`：说明如何用本体论盘点数据资产，并接入元数据字段、中文名、业务含义和备注信息的自动补全。
- `一键智能填充原型/Axure原型/`：Axure RP、draw.io、HTML、SVG、PNG 和 Axure 规格文档等既有原型交付物。
- `一键智能填充原型/Figma原型/`：Figma 可直接导入并继续编辑的四张独立 SVG 画板、全流程画板及 Prototype 连线说明。
- `ontology_metadata_model_example.json`：可机器读取的语义模型样例，展示实体、属性、关系、规则、血缘和权限如何结构化表达。

## 使用场景

- 数据治理方案设计
- 元数据管理平台建设
- 指标体系和语义层建设
- 智能问数、Text-to-SQL、RAG 和知识图谱应用
- 面向 AI 的企业数据资产梳理
